Question-1: What are the different ways to select an element in the DOM?

There are many way to select an element in JavaScript ,however those are | Document.getElementByTagName('tag name'), Document.getElementsByTagName('class name'), Document.getElementByTagId('id name'), Document.querySelector('#id name/.class name'), Document.querySelectorAll(#id name/.class name')| this are commonly use in js to select element
